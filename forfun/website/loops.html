for loop:

the for loop iterates(goes through each element) in a range.

for example, if i had a list like [1,2,3] called list and wrote something like:

for element in list: print('doodoo')

it would print 'doodoo' three times like so:

doodoo doodoo doodoo

here, the for loop is looking at each element one-by-one in list, and printing 'doodoo' one time for every element that it scans/detects/sees. the for loop assigns each element(object/thing) in the list a variable called element as i specified in the loop and does what it needs to do with it before continuing to iterate.

while loop:

the while loop keeps looping until a condition is reached.

for example, if i assigned 1 to a variable called x and wrote something like:

while x <= 3: print('yeah') x += 1

it would print 'yeah' three times like so:

yeah yeah yeah

here, the while loop is checking to see if x is smaller than or equal to 3. if it is, it prints 'yeah' and then adds one to the value of x. once the loop eventually assigns x to a value larger than three, it stops.

nested loop:

a nested loop is just a loop in another loop, like:

while *whatever condition*: for *whatever condition*: *something happens*

here, the for loop is the nested loop.
